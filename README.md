## 实验目的

1. 掌握共射电路静态工作点的计算、仿真、测试方法。
2. 掌握电路主要参数的计算、中频时输入、输出波形的相位关系、失真的类型及产生的原因。
3. 掌握获得波特图的测试、仿真方法。
4. 掌握负反馈对增益、上下限截频的影响，了解输入输出间的电容对上限截频的影响等。
